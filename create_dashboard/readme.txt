Steps of creating a dashboard:
1.go to 'dashboard module maintenance' to create a module.
2.go to dashboard screen, use add to add a tab.
3.in the tab, click add to add a moduled defined.

csv data note:
1.dash_tab_mod_config.description: define the title of the tab.
2.dash_tab_mod_config.refresh_sec: seconds for refreshing the dashboard.
3.dash_tab_assign.ath_typ: U-Update, R-Readonly
4.dash_mod_def.dash_dda_id:The dda ID used for the dashboard, if the dda type is pie/bar chart, then it will be chart displayed, otherwise it can be a grid.
