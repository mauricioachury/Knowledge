Readme:
1. dda_smart_pick.tar.gz
Contains DDA definition for smart picking, it can be accessed from:
a.) Login RF.
b.) 1 Undirected picking.
c.) 8 Smart picking.

2. MTF-DDA-Engine.zip
The engine code for converting DDA into RF form.

3. SmartPick-CSV-Files.zip
CSV records defined menus, poldat, error message etc.

4.SmartPick-MCMD-Files.zip
mcmd files for the Smart picking feature.

This is implemented in 9.1.4 version WMS.
